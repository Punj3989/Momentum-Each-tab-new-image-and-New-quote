# 🌅 Momentum - Har Naya Tab

> *"Har naya tab mein beautiful wallpaper + motivation"*

A Chrome extension that turns your boring new tab page into a beautiful, inspiring dashboard with stunning wallpapers, motivational quotes, live clock, and a daily focus tracker.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🖼️ **Beautiful Wallpapers** | Random HD images from Unsplash on every new tab |
| 💬 **Motivational Quotes** | 50+ inspiring quotes in Hindi & English |
| 🕐 **Live Clock** | Real-time clock with greeting (Good Morning/Afternoon/Evening/Night) |
| 📅 **Date Display** | Current date with day, month, and year |
| 🎯 **Daily Focus** | Set your main focus for the day (saved in localStorage) |
| 🌐 **Bilingual** | Supports both Hindi and English quotes |
| 📱 **Responsive** | Works perfectly on all screen sizes |
| 🎨 **Smooth Animations** | Fade-in effects and smooth transitions |
